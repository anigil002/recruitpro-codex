RecruitPro ATS

Human-in-the-loop AI recruiting platform.

Quick Start

  1. Run uvicorn app.main:app --reload.
  2. Use the endpoints documented in recruitpro_system_v2.5.md.
  3. Explore the API via the built-in Swagger UI at /docs.

API Endpoint Summary

The backend implements the 54 documented endpoints with pragmatic stub behavior suitable for local demos. Activity tracking and AI helpers use deterministic fallback data so the platform runs fully offline.
